### PIVOT TABLE



Act as an automated table for most of the purposes like filtering , sorting , counting etc



* How to create a new PIVOT TABLE

-Go in insert section

-Select the pivot table button

-Select the required Data for pivot table

-Select whether you want the pivot table in new sheet or the existing sheet , click enter





* How to create separate sheet of each filtered result in pivot table

-first add on some values in pivot table and select a filter there

-Go in pivot table/analyze section and look for pivot table name section

-In pivot table name click the options dropdown a dialogue box would appear showing all your applied filters , select the required filter and click ok

-New sheets will appear with separated filtered data



* How to create a pivot chart

-Select the required data table

-Go in insert section and select pivotChart option

-Select the required chart



* Changing the chart from one to another without creating a new one

-Select the current chart , go in design section

-Click change chart type option , sleect the required chart



* How to add explicit filter(without the use of pivot table filter section)

-Go in pivotable/analyze

-Go in Slicer and select the required filter
